# Fyn SDK

The offical SDK of the Fyn API.

## Generation

The sdk is generated in typescript using [Swagger CodeGen](https://github.com/swagger-api/swagger-codegen).
Each time a push is made on the [main branch of the api repository](https://github.com/Fyn-JobBoard/api/tree/main), the [sdk workflow](./.github/workflows/sdk.yaml) is triggered to generate the api from its swagger document.
